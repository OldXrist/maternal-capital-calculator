# Калькулятор долей в жилом помещении приобретенном с использованием средств материнского капитала

Десктопное приложение для расчёта долей супругов и детей в квартире, приобретённой с использованием материнского капитала в соответствии с российским законодательством. Разработано на PyQt6, обеспечивает равные доли родителей и детей, с округлением вверх долей детей по нормам закона.

## Возможности
- Ввод стоимости квартиры, использованного материнского капитала и количества детей (0–30).
- Опциональные ФИО родителей и детей (динамические поля для детей).
- Расчёт:
  - Долей супругов без учёта маткапитала.
  - Общих долей с маткапиталом.
- Отображение результатов в виде долей от 1000.
- Визуальная диаграмма в виде круговой (в %).
- Интерфейс на русском языке для использования в юридической практике.
- Обработка ошибок для некорректных входных данных.

## Требования
- Python 3.8+.

## Установка
1. Клонируйте или скачайте проект.
2. Установите зависимости: `pip install -r requirements.txt` (создайте файл, если нужно, с указанными выше пакетами).
3. Запустите: `python main.py`.

## Использование
1. Введите стоимость квартиры (руб.) и использованный материнский капитал.
2. Укажите количество детей — поля для имён появятся динамически.
3. Опционально добавьте ФИО родителей и детей.
4. Нажмите «Рассчитать доли».
5. Просмотрите текстовые результаты и круговую диаграмму.

## Сборка исполняемого файла
Используйте PyInstaller для создания автономного .exe:
1. `pip install pyinstallerр`
2. `pyinstaller --onefile --windowed --name=MaternalCalculator main.py`
